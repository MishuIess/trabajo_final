## Trabajo Fina- Analisis de datos de la salud mental.

El presente trabajo final tiene como finalidad analizar los datos del archivo . csv y .sqlite. dichos datasets la fuente es Kaggle.

Las herramientas utilizadas para el trabajo final es pandas, SQLite, Matplotlib y Google Colab.

Descripción de los pasos realizados en el proyecto: 

1.- Importación de librerías necesarias como Pandas, Matplotlib y SQLite.

2.- Carga del dataset CSV utilizando la función read_csv().

3.- Conexión a la base de datos SQLite mediante sqlite.connect().

4.- Lectura de información desde la tabla Answer utilizando consultas SQL.

5.- Exploración de los datos mediante funciones como head(), info() y describe().

6.- Limpieza de datos eliminando registros nulos.

7.- Integración de ambos datasets utilizando la función merge() de Pandas.

8.- Generación de visualizaciones estadísticas con Matplotlib.

9.- Exportación del dataset final procesado.

10.- Organización y subida de archivos al repositorio GitHub.
